Hi researchers! If you've ever struggled with non-conjugate Bayesian models where standard variational Bayes fails, this guide introduces **VB-LD** - a robust combination of Laplace approximations and Delta methods that maintains scalability while handling tricky parameter relationships.

## Why VB-LD Matters

Traditional variational Bayes shines with conjugate models but hits roadblocks when:
- Your prior isn't conjugate to the likelihood
- Posterior distributions become analytically intractable
- Importance sampling leads to unstable estimates

VB-LD solves these by combining:
1. **Laplace approximations** for non-conjugate parameter updates
2. **Delta methods** for moment estimation
3. **Mean-field factorization** to preserve computational efficiency

<div class="yellow-box">
  <strong>When to Use VB-LD:</strong>  
  <ul>
    <li>Hierarchical models with mixed conjugate/non-conjugate components</li>
    <li>Generalized linear models with custom link functions</li>
    <li>Latent variable models where exact updates are unavailable</li>
    <li>High-dimensional settings where MCMC is too slow</li>
  </ul>
</div>

## Core Concepts Explained

### 1. Mean-Field Variational Bayes (MFVB)

MFVB approximates the true posterior $p(\boldsymbol{\theta}|D)$ where $\boldsymbol{\theta} = (\boldsymbol{\eta}, \boldsymbol{\psi})$ with a factorized distribution:

```math
q(\boldsymbol{\theta}) = q(\boldsymbol{\eta})q(\boldsymbol{\psi})
```

We optimize this by maximizing the **Evidence Lower Bound (ELBO)**:

```math
\elbo(q) = \mathbb{E}_q[\log p(D,\boldsymbol{\theta})] - \mathbb{E}_q[\log q(\boldsymbol{\theta})]
```

<div class="green-box">
  <strong>Key Insight:</strong>  
  This factorization turns a high-dimensional integration problem into simpler, iterative updates of $q(\boldsymbol{\eta})$ and $q(\boldsymbol{\psi})$.
</div>

### 2. The Non-Conjugate Challenge

For non-conjugate $\boldsymbol{\eta}$, the VB update:

```math
q(\boldsymbol{\eta}) \propto \exp\left(\mathbb{E}_{q(\boldsymbol{\psi})}[\log p(D,\boldsymbol{\eta},\boldsymbol{\psi})]\right)
```

faces three fundamental issues:
1. **Intractable normalization** - Can't compute the integrating constant
2. **Unavailable moments** - $\mathbb{E}_{q(\boldsymbol{\eta})}[g(\boldsymbol{\eta})]$ lacks closed form
3. **ELBO evaluation** - Can't assess convergence

## The VB-LD Solution

### 1. Laplace Approximation Step

For non-conjugate $\boldsymbol{\eta}$, we:

1. **Find the mode** via Newton-Raphson:
```math
\boldsymbol{\eta}^{(k+1)} = \boldsymbol{\eta}^{(k)} - [\nabla^2 f(\boldsymbol{\eta}^{(k)})]^{-1}\nabla f(\boldsymbol{\eta}^{(k)})
```
where $f(\boldsymbol{\eta}) = \mathbb{E}_{q(\boldsymbol{\psi})}[\log p(D,\boldsymbol{\eta},\boldsymbol{\psi})]$

2. **Approximate covariance**:
```math
\Sigma = -[\nabla^2 f(\hat{\boldsymbol{\eta}})]^{-1}
```

3. **Set variational distribution**:
```math
q(\boldsymbol{\eta}) \approx \mathcal{N}(\hat{\boldsymbol{\eta}}, \Sigma)
```

<div class="yellow-box">
  <strong>Pro Tip:</strong>  
  The Laplace approximation is exact as $n \to \infty$ for regular models (Bernstein-von Mises theorem). Monitor the Hessian's positive-definiteness to ensure validity.
</div>

### 2. Delta Method for Expectations

To update $q(\boldsymbol{\psi})$, we often need $\mathbb{E}[g(\boldsymbol{\eta})]$. The Delta method provides:

```math
\mathbb{E}[g(\boldsymbol{\eta})] \approx g(\hat{\boldsymbol{\eta}}) + \frac{1}{2}\text{tr}[\nabla^2 g(\hat{\boldsymbol{\eta}})\Sigma]
```

Common applications:
- **First moments**: $g(\boldsymbol{\eta}) = \eta_i$
- **Second moments**: $g(\boldsymbol{\eta}) = \eta_i\eta_j$
- **Transforms**: $g(\boldsymbol{\eta}) = \log \eta_i$

## Complete VB-LD Algorithm

```python
# Pseudocode implementation
def VB_LD(model, data, max_iter=100, tol=1e-6):
    # Initialize
    q_psi = init_variational(model.conjugate_params)
    elbo_trace = []
    
    for iter in range(max_iter):
        # Laplace step
        eta_hat, Sigma = laplace_approximation(model, data, q_psi)
        q_eta = Gaussian(eta_hat, Sigma)
        
        # Delta step
        moments = delta_method(model, eta_hat, Sigma)
        
        # Conjugate step
        q_psi = update_conjugate(model, data, moments)
        
        # Convergence check
        elbo = compute_elbo(model, q_eta, q_psi)
        elbo_trace.append(elbo)
        if converged(elbo_trace, tol):
            break
            
    return q_eta, q_psi, elbo_trace
```

## Practical Considerations

### When VB-LD Excels
- Models with **few non-conjugate parameters** but many conjugate ones
- **Unimodal posteriors** where Laplace approximation is accurate
- **Medium-to-large datasets** where MCMC is prohibitively slow

### Limitations to Watch For
- **Multimodal distributions** - Laplace approximates only one mode
- **High-dimensional $\boldsymbol{\eta}$** - Hessian computation becomes expensive
- **Skewed posteriors** - Gaussian approximation may be poor

<div class="red-box">
  <strong>Debugging Tip:</strong>  
  Always compare VB-LD results with:
  <ul>
    <li>MCMC when feasible</li>
    <li>Simpler conjugate versions of your model</li>
    <li>Synthetic data where ground truth is known</li>
  </ul>
</div>
